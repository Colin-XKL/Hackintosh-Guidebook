# Hackintosh Guidebook
[//]: <> (Project README) 
黑苹果快速入门手册，为有一定基础的黑苹果新人提供指引
## Description
本项目旨在为有一定基础的黑苹果新人（至少懂分区和重装系统），提供一些指引，让更多有兴趣且有能力的朋友吃上黑苹果

*疫情期间，人人居家，狂肝黑苹果*
